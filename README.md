## ✨ Features

- 🔍 Search trains based on source and destination
- 🚆 Dynamic train listing
- 📝 Passenger details form
- 🎫 Automatic PNR generation
- 💺 Automatic seat allocation
- 📜 Booking history management
- ❌ Ticket cancellation functionality
- 💾 Local Storage based data persistence
- 📱 Fully responsive design
- 🎨 Clean and modern UI inspired by IRCTC

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/shhivamverma/Railway-Booking-System.git
cd Railway-Booking-System
```

### Run Locally

Simply open:

```text
index.html
```

or run using **Live Server** in VS Code.

## 🎯 Core Functionalities

### 🔍 Train Search
Users can search for trains using:
- Source station
- Destination station
- Journey date
- Travel class

### 🎫 Ticket Booking
- Passenger information form
- Dynamic seat assignment
- Unique PNR generation

### 📜 Booking History
- View all booked tickets
- Cancel existing bookings
- Persistent data using Local Storage
